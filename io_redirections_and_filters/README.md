# Shell — I/O Redirections and Filters

## Tasks

### 0. Hello World — `0-hello_world`

Prints **"Hello, World"**.

### 1. Confused smiley — `1-confused_smiley`

Displays `"(Ôo)'`.

### 2. Display a file — `2-hellofile`

Shows content of `/etc/passwd`.

### 3. Display two files — `3-twofiles`

Shows `/etc/passwd` and `/etc/hosts`.

### 4. Last lines — `4-lastlines`

Displays last 10 lines of `/etc/passwd`.

### 5. First lines — `5-firstlines`

Displays first 10 lines of `/etc/passwd`.

### 6. Line #3 — `6-third_line`

Displays the 3rd line of `iacta`.

### 7. Special file — `7-file`

Creates a file with special name containing `Best School`.

### 8. Save directory state — `8-cwd_state`

Saves `ls -la` output into `ls_cwd_content`.

### 9. Duplicate last line — `9-duplicate_last_line`

Repeats the last line of `iacta`.

### 10. Remove .js — `10-no_more_js`

Deletes all `.js` files recursively.

### 11. Count directories — `11-directories`

Counts all directories (including hidden).

### 12. Newest files — `12-newest_files`

Lists 10 newest files.

### 13. Unique — `13-unique`

Prints unique words, sorted.

### 14. Find "root" — `14-findthatword`

Displays lines with `root` from `/etc/passwd`.

### 15. Count "bin" — `15-countthatword`

Counts lines with `bin` in `/etc/passwd`.

### 16. What’s next — `16-whatsnext`

Shows lines with `root` and 3 after.

### 17. Hide "bin" — `17-hidethisword`

Displays lines without `bin`.

### 18. Letters only — `18-letteronly`

Shows lines starting with a letter from `sshd_config`.

### 19. A to Z — `19-AZ`

Replaces `A→Z` and `c→e`.

### 20. Without C — `20-hiago`

Removes `c` and `C`.

### 21. Reverse — `21-reverse`

Reverses input.

### 22. Users and homes — `22-users_and_homes`

Displays `user:home` list sorted by user.
